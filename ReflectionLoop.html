How to Use ReflectionLoop

Welcome to ReflectionLoop!

This tool helps you find and remove private information from your text before sharing it.

Step 1: Enter Your Text

• Copy and paste your text into the "Input Text" box
  - or click "Upload TXT" to load a .TXT or.CSV file

Step 2: Find Private Information

• Click the "Privacy Scan" button (it will be pulsing/glowing)
• The tool will scan for SSN, credit cards, emails, phone numbers, etc.
• Found items will be highlighted in the bottom box
• The "Clean Text" button will start pulsing when detections are found

Step 3: Review What Was Found

• Look at the highlighted items
• Border colors show confidence:
  - Red   - Orange   - Blue border = Low confidence (pattern only)
• Hover over highlights to see confidence level
• Click any highlight to KEEP that information (it turns gray)

Step 4: Clean Your Text

• Click "Clean Text" (it will be pulsing)
• Private information becomes [REDACTED] placeholders
• You can edit the text after cleaning
• Both "Copy Clean" and "Clear All" buttons will start pulsing

Step 5: Copy Your Clean Text

• Click "Copy Clean" (pulsing) to copy bold [REDACTED] text
  - Use "Clear All" (pulsing) to start over
  - The Clear All button will continue pulsing until clicked
• "Clear All" to activate new "Privacy Scan"

Copy Formatting Guide (Chrome/Edge):

Keep highlighting and bold redactions when pasting to Word/Writer:
• From Detection Results: CTRL+A CTRL+C and paste into your document for yellow highlighting
• "Copy Clean" button: Preserves bold [REDACTED] placeholders

Windows Copy HIGHLIGHTED Steps:
• Click in the Detection Results after "Privacy Scan"
• Press CTRL+A (select all text)
• Press CTRL+C (copy with HIGHLIGHTED formatting)
• Open Microsoft Word or LibreOffice Writer document
• Press CTRL+V (paste with HIGHLIGHTED formatting)


Copy [REDACTED] Steps:
• Select "Privacy Scan"
• Select "Clean Text"
• Select "Copy Clean"
• Open Microsoft Word or LibreOffice Writer
• Press CTRL+V (paste with [REDACTED] formatting)

What transfers to Word/Writer:
• Yellow highlighting from Detection Results
• Red borders for high confidence items
• Orange borders for medium confidence items
• Blue borders for low confidence items
• [REDACTED] text appears bold in Word/Writer documents

Pro Tip: Use manual copy (CTRL+A, CTRL+C) instead of "Copy Clean" button for preservation of highlighted text.

Firefox only: Redacted bold copy/paste works, highlight copy/paste does not. For highlight copy/paste, use Chrome or Edge.

Understanding Confidence Scores:

What makes detection confidence high?
• High: Keywords found nearby (e.g., "SSN:" before numbers)
• Medium: Some context present (e.g., "call" near phone pattern)
• Low: Pattern matches but no supporting context

Statistics show:
• Total detections and breakdown by confidence
• Higher confidence = more likely to be sensitive data
• Review low confidence items carefully - they might be false positives

Pulsing Button Guide:

The buttons will pulse/glow to guide you through each step:
• Privacy Scan pulses when you enter text
• Clean Text pulses after detection finds PII/PHI
• Copy Clean and Clear All pulse after "Privacy Scan"
• Follow the pulsing buttons for the easiest workflow!

Export Report Guide:

To download files click "Export Report" for Export Options:
• Word/Writer Document automatic download .docx file format
• PDF Document print to .pdf, not automatic for security
• CSV Report automatic download .csv file format
• Text File automatic download .txt file format
  -If "Export Report" before "Clean Text" exports auto redacted

File Name Guide:

Downloaded files will be automatically named except PDF:
• Word Document .docx CURRENT DATE ReflectionLoop - Report
• PDF Document .pdf = user names
• CSV Report .csv CURRENT DATE ReflectionLoop - Report
• Text File .txt CURRENT DATE ReflectionLoop - Report

Privacy Timer Guide:

Privacy Timer and Screen Blur information:
• Privacy Timer starts after "Privacy Scan" finds data
• Screen Blur every 45 seconds inactive session, move mouse or touch screen
• Privacy Timer and Screen Blur designed for security
  - Timer countdown 5 minutes, then auto-clears data for security.
  - Blur and data clear together when timer expires.

Performance Guidelines:

Document Size Recommendations:

Ideal Performance (Field Laptops):
• Up to 10,000 words (≈23 pages single-spaced)
• ~250 detections typical
• Smooth operation on all devices

Good Performance:
• Up to 25,000 words (≈58 pages single-spaced)
• ~600 detections typical
• Minor lag only during export

Reduced Performance:
• 50,000-100,000 words (115-230 pages single-spaced)
• 1,200-2,500 detections typical
• Noticeable lag, use "Copy Clean" for best results

Pro Tip: Documents over 25,000 words, split into sections for optimal review performance.

Tested on: Intel i5 @ 2.40GHz, Intel(R) Iris(R) Xe Graphics, 32GB RAM
Browsers Tested: Chrome, Edge, Firefox (no console errors)
